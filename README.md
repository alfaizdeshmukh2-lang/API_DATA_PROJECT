# OVERVIEW
The api data project is a python based application that fetches data from multiple public REST APIs, process the response, display structured output in the terminal and save the data into JSON file.
The project follows modular programing , demonstrate real world API handling.

# FEATURES 
1) Fatches data from Github public API
2) Fetches data from JSON PLACEHOLDER post API
3) Display formated output in the terminal 
4) Saves API responses into JSON file 
5) Clean, reusable, modular code structure 


# Technology Used
1) python 3
2) requests library 
3) REST APIs
4) JSON
5) File Handling 

# API USED 
- GitHub API : "https://api.github.com/users/{username}/repos"
- JSONPlaceholder Post API : "https://jsonplaceholder.typicode.com/posts"
